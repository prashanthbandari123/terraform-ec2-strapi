# terraform-ec2-strapi

EC2 Provisioning & Strapi Deployment Using Terraform

Launched an EC2 instance manually via AWS Console.

Verified SSH access using a secure key pair.

Provisioned an EC2 instance using Terraform (module-based).

Created and configured an SSH key pair for Terraform deployment.

Configured Security Group rules for SSH (22) and Strapi (1337).

Installed and deployed the Strapi application on the EC2 server.

Verified Strapi is running successfully and accessible in a browser.
